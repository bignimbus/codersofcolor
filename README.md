## Setup

 <pre><code>
 $ git clone https://github.com/keithgordon/codersofcolor.git
 $ cd codersofcolor
 $ bundle install
 $ rake db:migrate
 $ rails server
 <pre></code>

<p>navigate to http://localhost:8080/</p>
